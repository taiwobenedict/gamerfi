INTRODUCTION

The GAMERFI studio is introducing a new token to support in-app purchases for all the games in its portfolio, including its own games and partner games. A central app will be created to list all supported games, similar to the Google Play Store, where GMI token users can browse and download the different games available.

Whitepaper

KEY FEATURES

In-app purchases: The GMI token will support in-app purchases for all games under the GAMERFI portfolio, allowing users to buy virtual goods and items within the games. A central app will be created to list all games that are supported by the GMI tokens, allowing users to easily browse and download games. Token support: The GMI token will be used to support in-app purchases, providing a seamless and secure payment solution for users.
